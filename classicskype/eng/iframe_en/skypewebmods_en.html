Mods of the old Web Client

Here you can try two key web client mods. Both are under development, so some features may be missing.
   TrySkype

The first public client. Based on Skype for Linux Beta (Skype Web Wrapper). It is under development, has its own servers, from where the necessary scripts are already loaded. Can't call but it will be soon.
Download (Discord Server)
Download (TrySkype Website)
   Classic Skype by vl0d

Based on Skype Web Wrapper
It is constantly evolving. Unlike TrySkype, it connects only to official Microsoft servers, modifications are performed only on the client side. It is possible to call, chats from 2017 are visible and many other good features
Download (Discord Server)
