Метод 2 - Просматриваем интерфейс при помощи Window в Process Hacker 2

Метод заключается в том, чтобы просто включить нужные нам окна через программу Process Hacker 2

 

1. Запускаем Skype. Данный трюк можно провернуть с версиями 1.x / 2.x / 3.x / 5.x / 6.x / 7.x

4.x НЕ ПОДДЕРЖИВАЕТСЯ!!!

2. На окне логина открываем Process Hacker 2, ищем процесс скайпа

3. ПКМ по нужному процессу -> miscellaneous -> windows

4. В открывшейся менюшке просто эксперементируем, включая разные состояния окна нажатием ПКМ -> visible

Right click on picture -> Open Image in new tab for Full Quality
